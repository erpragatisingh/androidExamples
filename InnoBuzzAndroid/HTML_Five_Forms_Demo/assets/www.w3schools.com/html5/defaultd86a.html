HTML5 Multimedia

With HTML5, playing video and audio is easier than ever.

  • HTML5 <video>
  • HTML5 <audio>

HTML5 Applications

With HTML5, web application development is easier than ever.

  • Local data storage
  • Local file access
  • Local SQL database
  • Application cache
  • Javascript workers
  • XHTMLHttpRequest 2

HTML5 Graphics

With HTML5, drawing graphics is easier than ever:

  • Using the <canvas> element
  • Using inline SVG
  • Using CSS3 2D/3D
